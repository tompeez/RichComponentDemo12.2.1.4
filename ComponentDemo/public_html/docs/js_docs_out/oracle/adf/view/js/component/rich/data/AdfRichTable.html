getRowIndex

public Number getRowIndex(String rowKey)

Returns the index of a row identified by rowKey.
There is no guarantee that a row identified by a rowKey can be found on the client since all the rows are not rendered on the client. Also keep in mind that the rowKey on the client is not the same as the one on the server, since the server provides a mechanism for converting client row keys into server row keys.

Parameters:
rowKey  -  the client side rowKey that identifies a row
Return:
Number - the index of a row for a row key if is found otherwise -1.
See also:

simulateSelectRows

public Object simulateSelectRows(String rowKeyList,
                                 String operation)

Simulate a user selecting one or more rows from the user interface.
If more than one row is specified, it is up to the test to make sure the component allows multiple row selection.
Valid values for operation are:
"replace" - replace current selection with the rows specified by rowKeyArray.
"toggle" - toggle the selection state of the rows specified by rowKeyArray.


Parameters:
rowKeyList  -  - a comma-separated list of rowkeys that specifies the rows to operate on
operation  -  - the operation to perform. "replace" is the default if no valid value is passed for this parameter.
Return:
Object
