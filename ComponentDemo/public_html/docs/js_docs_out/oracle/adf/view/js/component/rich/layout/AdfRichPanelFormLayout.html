public class AdfRichPanelFormLayout
extends AdfUIPanel
The panelFormLayout positions input form controls, such that their labels and fields line up vertically. It supports multiple columns, and also supports a footer facet. When multiple columns are present, there is no guarantee of alignment across columns (if this is the kind of structure you need, you should use some kind of grid component instead).
