public class AdfDeckSwipeManager
extends AdfObject
Helper class to manage swipe and pan actions for the deck component. This class manages two panels - the visible content panel and a loading-indicator/swipe-in-progress panel. As the user swipes the screen using touch gesture, both contentPanel and swipe-loading panel slide across the screen. A displayChange event is queued on swipe gesture, on response of which the loading indicator fades away showing the updated contentPanel. On cancelling of event or validation failures, the contentPanel and swipe-loading panel transition in the reverse direction as the user swiped to restore the UI in the condition it was earlier.
