public class AdfConveyorBeltVertical
extends AdfConveyorBeltSupport
Support class for handling vertical conveyor belt layout for a component. Note that if you use AdfConveyorBeltVertical, you will: Need to make sure your container has at least 1 element. Need to call removeConveyorBeltSupport from your component's ComponentRemoved method. Have the direct children of the conveyor belt container use the p_AFSelected style to mark selection.
