public AdfCalendarEvent(Object source,

                        Number triggerMin,
                        Object keyStroke,
                        Object clickCount,
                        Object button,
                        Object triggerType)

Calendar Event class

Parameters:
source  -  Source of the CalendarActivity event
triggerDate  -  The date string (e.g., 07/31/2008) of the calendar's day area where the user clicked.
triggerHour  -  The hour of the calendar's free space area where the user clicked. This will be null if the user clicked in the all-day free space area.
triggerMin  -  The minute of the calendar's free space area where the user clicked. This will be null if the user clicked in the all-day free space area.
keyStroke  -  AdfKeyStroke object with the keystroke information that triggered the event.
clickCount  -  number of button clicks: 0, 1, 2
button  -  "NONE", "LEFT", or "RIGHT"
triggerType  -  "MOUSE", "FOCUS", "KEY", "HOVER", "CONTEXT_MENU" TODO Do I need an enum for the client api as well? AdfCalendarEventType and AdfCalendarEventButton
