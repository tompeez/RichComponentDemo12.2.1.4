_growTransition

private static Object _growTransition(Object rootElement,
                                      Object wrapper,
                                      Object styleProperties,
                                      Object anchor)

This is default behavior to show popup animation as growing diagonally. The diagonal growing direction will be based on two parameters, i.e. anchor.isRight, anchor.isBottom. The actual popup, rootElemnt, will not resize or animate, it will float in the wrapper element. The root popup element will stick to two of the wrapper div edges, based on anchor.isRight and anchor.isBottom values. Doing so, the root element will be positioned along with wrapper elements. The approach is, first set the root element position styles, then initial wrapper animation properties, from where animation will start, and at last compute animation properties, where wrapper animation will end. return properties, will be used by AdfDhtmlElementAnimator.animate() to make sliding frams, to show animation.

Parameters:
rootElement    
wrapper    
styleProperties    
anchor

_horizontalSlideTransition

private static Object _horizontalSlideTransition(Object rootStyle,
                                                 Object wrapperStyle,
                                                 Object styleProperties,
                                                 Object isOpening,
                                                 Object slideDirection)

To make horizontal sliding animation. The horizontal sliding direction will be based on two parameters, i.e. isOpening, slideDirection. The actual popup, rootStyle, will not resize or animate, it will float in the wrapper element. The root popup element will stick to two of the wrapper div edges, based on isOpening and slideDirection values. Doing so, the root element will be positioned along with wrapper element sliding animation. The approach is, first set the root element position, then set the initial wrapper animation properties, from where animation will start, and compute animation properties, where wrapper animation will end. return properties, will be used by AdfDhtmlElementAnimator.animate() to make sliding frams, to show animation.

Checks to see if the "ancestorNode" is a ancestor of "node" or if they are the same. This is a popup specific version of "AdfDomUtils.isAncestorOrSelf" that jumps back to the root dom element of a open popup so that it reflects back to the origianl dom structure before a popup is open. To make vertical sliding animation. The vertical sliding direction will be based on two parameters, i.e. isOpening, slideDirection.
