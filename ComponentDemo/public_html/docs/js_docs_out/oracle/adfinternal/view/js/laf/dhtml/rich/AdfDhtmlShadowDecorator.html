public class AdfDhtmlShadowDecorator
extends AdfObject
AdfDhtmlShadowDecorator is used to decorate a DOM element with shadow. This object basically creates transparent div elements underneath the DOM element for the shadows. This object also creates a iframe underneath to clip any OS drawn controls

Creates the DOM elements for the shadow associated with a given element. NOTE: There is no back pointer to the DOM element to avoid memory leaks in IE

Parameters:

Default value = 3
shadowOffsetLeft  -  The x offset of the shadow, default value is 3 if null is provided
Default value = 3
isRelative  -  Indicates if the shadow is relatively positioned.
startingBlackPercent  -  The starting black percent level for the shade (effects darkness/lighness of drop shadow).
Default value = 100
