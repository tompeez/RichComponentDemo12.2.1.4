private static Object _SUB_ID_MAP

panelTabbed component publishes the following sub ids [External Subid --> Internal Client SubId]:- 1. [top_start_overflow_indicator --> panelTabbed top header start overflow indicator (mapped programmatically)] 2. [top_end_overflow_indicator --> panelTabbed top header end overflow indicator (mapped programmatically)] 3. [bottom_start_overflow_indicator --> panelTabbed bottom footer start overflow indicator (mapped programmatically)] 4. [bottom_end_overflow_indicator --> panelTabbed bottom footer end overflow indicator (mapped programmatically)] 5. [top_start_conveyor_belt_indicator --> panelTabbed top header start conveyor belt indicator (mapped programmatically)] 6. [top_end_conveyor_belt_indicator --> panelTabbed top header end conveyor belt indicator (mapped programmatically)] 7. [top_conveyor_belt_droplist_indicator --> panelTabbed top header conveyor belt droplist indicator (mapped programmatically)] 8. [bottom_start_conveyor_belt_indicator --> panelTabbed bottom footer start conveyor belt indicator (mapped programmatically)] 9. [bottom_end_conveyor_belt_indicator --> panelTabbed bottom footer end conveyor belt indicator (mapped programmatically)] 10. [bottom_conveyor_belt_droplist_indicator --> panelTabbed bottom footer conveyor belt droplist indicator (mapped programmatically)] 11. [top_conveyor_belt_indicator --> panelTabbed vertical top conveyor belt indicator (mapped programmatically)] 12. [bottom_conveyor_belt_indicator --> panelTabbed vertical bottom conveyor belt indicator (mapped programmatically)] 13. [toggle_tab_text_icon --> panelTabbed vertical text display indicator (mapped programmatically)]
