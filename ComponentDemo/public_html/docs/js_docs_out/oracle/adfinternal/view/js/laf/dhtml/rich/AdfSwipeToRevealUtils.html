public static Object handleDrag(Object component,
                                Object rootElement,
                                Object deltaX,
                                Object deltaY)

Drag handler to apply animations and effects required to produce a swipe gesture on list item. Renders actions inline with the list item based on how much distance user has swiped through the item. Also determines whether default action needs to be triggred on mouseup which again depends on how much distance user has swiped.
