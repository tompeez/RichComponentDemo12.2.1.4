getBestMatchSubIdForShowDetailItem

public static Object getBestMatchSubIdForShowDetailItem(Object childElement)

Called by a containing layout component peer (AdfDhtmlPanelAccordionPeerSubIds.js, AdfDhtmlPanelSpringBoardPeerSubIds.js,AdfDhtmlPanelTabbedPeerSubIds.js) and finds the best match subId of ShowDetailItem sub-elements if the childElement is a part of a ShowDetailItem. This special handling is needed for ShowDetailItem because the containing layout components renders subelement of owning ShowDetailItem outside of the ShowDetilItem component node.

Parameters:
childElement  -  the dom element
Return:
Object - best matched subid for the ShowDetailItem or null if the child element is not a show detail item or the sibid is not found

getSubIdDomElementForShowDetailItem

public static HTMLElement getSubIdDomElementForShowDetailItem(Object elementId)

Called by containing layout component peer (AdfDhtmlPanelAccordionPeerSubIds.js, AdfDhtmlPanelSpringBoardPeerSubIds.js,AdfDhtmlPanelTabbedPeerSubIds.js) and tries to find the DOM element that corresponds to the locator is passed in. It assumes the elementId passed in is an elementId for ShowDetailItem sub-element that has a format of <ShowDetailItem ClientID>#<SDI External SubId>, # otherwise returns null.

Parameters:
elementId  -  : id in the format <ShowDetailItem ClientID>#<SDI External SubId>
Return:
HTMLElement - a DOM element for a subElement that is belongs to the ShowDetailItem or null if none found.
