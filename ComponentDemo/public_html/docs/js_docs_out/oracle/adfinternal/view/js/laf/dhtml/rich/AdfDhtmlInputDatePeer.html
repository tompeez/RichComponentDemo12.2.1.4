_SUB_ID_MAP

private static Object _SUB_ID_MAP

inputDate component publishes the following sub ids [External Subid --> Internal Client SubId]:- 1. [timezoneName --> tzDispName] - Text element showing the selected timezone (localized). 2. [timezoneId --> tzDispId] - Hidden element containing the timezone id for the selected timezone. 3. [content --> content] - inputDate content 4. [pickerIcon --> glyph] - icon that launches the date picker

GetComponentLabel

protected String GetComponentLabel(Object component,
                                   Object domElement)

Lazy getter for the label property of the component. The default implementation AdfDhtmlEditableValuePeer does a DOM scan for the label elements and does not discriminate betweeen child label elements and actual component label with a matching "for". Hence the super implementation is susceptible to return the hidden labels from the month/year dropdowns in the picker, when the component is simple="true" and has no label attribute configured.

Parameters:
component  -  Component that this peer will be associated with
domElement  -  Root DOM element of this peer
Return:
String - label
