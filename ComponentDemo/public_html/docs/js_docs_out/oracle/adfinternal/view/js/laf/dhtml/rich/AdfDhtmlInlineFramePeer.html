_replaceIframe

private static frame _replaceIframe(Object iframeElement)

Pass in the initial iframe created for the component and creates a new iframe with the target source applied. The iframe is initally rendered with a blank.html src attribute. This is to avoid GET requests to the actual page during a PPR when the iframe is created. This sledgehammer approach avoids browser history for iframe navigation so that the browser back button doesn't record the navigation.

Parameters:
iframeElement    
Return:
frame - new iframe created with the target source
