<dvt:timeAxis>

dvt:timeAxis timeAxis time axis

UIComponent class: oracle.adf.view.faces.bi.component.gantt.TimeAxis
Component type: oracle.dss.adf.gantt.TimeAxis

The TimeAxis is used to specify the time axis in Gantt and Timeline components. When it is used in Gantt, the time axis component is used to specify the major and the minor time axis through the major and minor facets. When it is used in Timeline, the time axis component is used to specify the time axis for the timeline as well as the timeline overview.
