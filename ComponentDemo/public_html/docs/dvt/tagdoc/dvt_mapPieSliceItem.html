<dvt:mapPieSliceItem>

mapPieSliceItem map pie slice item


This tag is used when customizing the color of the slices in a particular <dvt:mapPieGraphTheme>. Each tag corresponds to one slice of the pie graph, and the color is specified using the color attribute.

In the example below, the map contains a pie graph theme where each graph has two slices--one with color=#0000ff and the other with color=#00ff00.

                          <dvt:map ...map attributes...>
                            <dvt:mapPieGraphTheme ...mapPieGraphTheme attributes...>
                              <dvt:mapPieSliceSet>
                                <dvt:mapPieSliceItem color="#0000ff" id="slice1"/>
                                <dvt:mapPieSliceItem color="#00ff00" id="slice2"/>
                              </dvt:mapPieSliceSet>
                            </dvt:mapPieGraphTheme>
                          </dvt:map>

Relationship with other tags

The <dvt:mapPieSliceItem> tag is a child of the <dvt:mapPieSliceSet> tag

Attributes

Name Type Supports EL? Description
id java.lang.String no Specifies the identifier for a slice on the pie charts of the mapPieGraphTheme
color java.lang.String yes Specifies the color of a slice on the pie charts of the mapPieGraphTheme
