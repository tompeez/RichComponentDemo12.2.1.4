<dvt:mapToolbar>

dvt:mapToolbar mapToolbar map toolbar

UIComponent class: oracle.adf.view.faces.bi.component.geoMap.UIGeoMapToolbar
Component type: oracle.dss.adf.geoMap.GeoMapToolbar

This component contains tools for turning on/off themes, selecting theme elements, zooming, calculating distance and area for the selected regions, displaying legend, and showing the spatial coordinates of the mouse location.

To use the map toolbar, you need to add the <dvt:mapToolbar> to your page, and set the mapId attribute to the id of the <dvt:map>. The mapToolbar tag is not a child tag of the map.

Map Tools

The mapToolbar contains the following set of tools:

  • "Pan" - Let users pan the map by drag and drop
  • "Zoom In" - Let users zoom in one level down the map
  • "Zoom Out" - Let users zoom out one level up the map
  • "Rectangle Tool" - Let users to make a rectangle selection on the map. Theme elements that are within the rectangle will be highlighted
  • "Circle Tool" - Let users to make a circle selection on the map. Theme elements that are within the circle will be highlighted
  • "Polygon Tool" - Let users to make a polygon selection on the map. Theme elements that are within the polygon will be highlighted
  • "Point Tool" - Let users to make a point selection on the map. Theme elements that are within the point will be highlighted
  • "Distance Tool" - Let users to draw a line and find the distance of the line
  • "Area Tool" - Let users to draw a polygon and find the area of the region
  • "Leged Tool" - Show/hide the legend
  • "Info Area Tool" - Show/hide the information area

Relationship with other tags

The <dvt:mapToolbar> tag is used in conjunction with the <dvt:map> tag.

Geometry Management

  • This component can be stretched horizontally by a parent layout component that stretches its children, e.g. panelStretchLayout, panelSplitter.
  • When NOT stretched, this component will be displayed in its default dimensions which are provided by the skin. This can be modified by applying a dimension using the inlineStyle or AFStretchWidth attributes.
  • This component does not stretch its children.
