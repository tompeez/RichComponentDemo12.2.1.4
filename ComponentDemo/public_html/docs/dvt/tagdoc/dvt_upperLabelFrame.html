E52773-01

<dvt:upperLabelFrame>

upperLabelFrame upper label frame


Use the upperLabelFrame tag to specify the colors for the frame behind the top label.

Note: UpperLabelFrame is not rendered unless the title separator is turned off.

Relationship with other tags

The upperLabelFrame tag is a child tag of the gauge tag.
The upperLabelFrame tag can have a child tag: specialEffects

Example

The following example shows the XML for the upperLabelFrame tag where the frame is rendered and both fill color and border color are specified.

   <dvt:gauge gaugeType="LED"> 
      <dvt:upperLabelFrame rendered="true" fillColor="#C0C0C0" borderColor="#0F0F0F"/>
   </dvt:gauge>

Attributes

Name Type Supports EL? Description
id java.lang.String no Specifies the identifier for the component
fillColor java.lang.String yes Specifies the fill color of the upper label frame. Enter values in RGB hexadecimal. The example color="#000000" specifies black color. This color property takes 6 or 8 digit hex as input. When 8 digit hex is used, the first 2 digits represent transparency, otherwise the color is opaque.
borderColor java.lang.String yes Specifies the border color of the upper label frame. Enter values in RGB hexadecimal. The example color="#000000" specifies black color. This color property takes 6 or 8 digit hex as input. When 8 digit hex is used, the first 2 digits represent transparency, otherwise the color is opaque.
rendered boolean yes Specifies whether the upper label frame is visible. Valid values are "true" or "false". The upper label frame is not displayed when the title separator is shown.
