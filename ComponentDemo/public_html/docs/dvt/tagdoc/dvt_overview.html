<dvt:overview>

dvt:overview overview overview

UIComponent class: oracle.adf.view.faces.bi.component.overview.UIOverview
Component type: oracle.dss.adf.overview.Overview

The Overview component provides a macro view of all the data points in the associated chart. Users can scroll through the chart using the Overview. The Overview component must be placed inside of a chart component.
