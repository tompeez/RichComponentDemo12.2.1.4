<dvt:showLegendGroup>

dvt:showLegendGroup showLegendGroup show legend group

UIComponent class: oracle.adf.view.faces.bi.component.legend.UIShowLegendGroup
Component type: oracle.dss.adf.legend.ShowLegendGroup

The showLegendGroup is a component which is used to display a disclosable area in a legend component. Its content can be shown and hidden by toggling a button next to its title. <dvt:legendSection> is a valid child tag of the showLegendGroup tag.

Example Usage

                    <dvt:legend id="l1" label="Legend Title">
                        <dvt:showLegendGroup id="slg1" label="Legend Subtitle 1">
                            <dvt:legendSection id="ls1" source="s1" />
                            <dvt:legendSection id="ls2" source="s2" />
                        </dvt:showLegendGroup>
                        <dvt:showLegendGroup id="slg2" label="Legend Subtitle 2">
                            <dvt:legendSection id="ls3" source="s3" />
                            <dvt:legendSection id="ls4" source="s4" />
                        </dvt:showLegendGroup>
                    </dvt:legend>

Relationship with other tags

The <dvt:showLegendGroup> tag can have the following children:

  • <dvt:legendSection>
