<dvt:mapBarSeriesSet>

mapBarSeriesSet map bar series set


This tag is used to wrap the <dvt:mapBarSeriesItem> tags that represent items of a <dvt:mapBarGraphTheme>

Note: The sequence of the mapBarSeriesItem tags is significant, and must reflect the sequence of data category items listed in the "Series Attribute" column of the data binding dialog for a mapBarTheme.

Relationship with other tags

The <dvt:mapBarSeriesSet> tag is a child of the <dvt:mapBarGraphTheme> tag

The <dvt:mapBarSeriesSet> tag can have the following children:

  • <dvt:mapBarSeriesItem>
